Welcome to the Types of Search Category
Hasty Search
• Requires a quick response

• Smaller teams are deployed

• Helps to define and shrink the search area

• Provides an immediate show of effort from searchers
Efficient Search
• Maximises probability of success

• Requires trained search resources

• Systematic searches using searcher spacing

• Usually assigned to sectors/segments/reflectors
Thorough Search
• Used as a last resort

• Very resource intensive

• Close line grid searching

• Maximum detection but very slow

• Can involve other agencies, such as the general public
The effectiveness of each type of search depends on the scenario.
